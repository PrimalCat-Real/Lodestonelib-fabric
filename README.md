# Lodestonelib-fabric

## FAQs
### How to install?
Add this in the gradle `repositories` block:
```groovy
maven { url 'https://maven.cph101.com/releases' }
```
...and this in the dependencies block:
```groovy
modImplementation 'team.lodestar.lodestone:Lodestonelib-fabric:0.0.4+1.20.1'
```
